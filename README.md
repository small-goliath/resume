# Small-goliath Resume

이 프로젝트는 **React**와 **Next.js**를 사용하여 개발된 자기소개 웹사이트입니다.


## 🚀 기술 스택

- **Frontend**: React 19, Next.js 15
- **Language**: TypeScript
- **Linting & Formatting**: ESLint
